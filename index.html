Waves Project

Waves Project aims to provide a big data platform which distributively processes RDF Stream. It has embedded lots of Big Data Framworks like Storm, Kafka, Spark, Docker and Redis etc. Thanks to an user-friendly graphical user interface, Waves is easily to be deployed and applied in a lot of usecases.
